Идея:
 - Управление ноутбуком с телефона.
 
Сокращенная идея (для диплома): 
 - Управление мышью ноутбука с андроида.

Принцип работы:
 1. Программа, запускаемая на ноутбуке (далее "ноутбук"), должна постоянно слушать входящие подключения, подразумевая, что к ноутбуку можно подключиться в любой момент после установки. 
 2. Программа, запускаемая на телефоне (далее "девайс", "телефон"), должен разослать всем участникам локальной сети сообщение о попытке подключиться. Далее список откликнувшихся будет отображен на экране девайса и пользователь сможет выбрать к кому именно он хочет подключиться для управления.
 3. Тем самым ноутбук должен отправлять в ответ имя своего пользователя для идентификации на девайсе.
 4. Пользователь может совершать определенные действия, которые будут отправлены и выполнены на ноутбуке:
	3.1. Движения тача по экрану соответствует движению курсора;
	3.2. Тап по экрану соответствует нажатию левой кнопки мыши;
	3.3. Двойной тап по экрану соответствует нажатию правой кнопки мыши;
 5. Отдельно следует вынести кнопки для управления медиа: изменение громкости и воспроизведение/пауза плеера.
 6. Опционально следует добавить отображение экрана ноутбука на телефоне.
