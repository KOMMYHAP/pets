Идея:
 - Управление ноутбуком с телефона.

Техническое задание:
1. Программа, запускаемая на ноутбуке (далее "ноутбук"), должна быть автономной и по большей мере не требовать участия пользователя, запускаться автоматически, быть компактной и не мешающей.
2. Программа, запускаемая на телефоне (далее "телефон"), должна отображать список устройств-ноутбуков в локальной сети, к которым можно выполнить подключение.
3. Пользователь с телефона может совершать определенные действия, которые будут транслироваться на ноутбук, таким образом для управления ноутбуком должно быть реализовано, как минимум:
	3.1. Работа с мышью:
		3.1.1. Движение мыши;
		3.1.2. Нажатие кнопок мыши;
		3.1.3. Скролл
	3.2. Работа с медиа:
		3.2.1. Изменение громкости;
		3.2.2. Пауза/воспроизведение плеера;
	3.3. Визуальная составляющая:
		3.3.1. Отображение картинки с экрана ноутбука;
