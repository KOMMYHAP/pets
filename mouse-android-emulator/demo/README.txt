1 Java Native Interface (jni) - интерфейс для вызова плюсового (нейтив) кода из java кода. Позволяет переиспользовать уже готовый код на плюсах (нашу мини-серверную архитектуру) и запускать его же на мобилке.
2. Google Protobuf - библиотека для удобной работы с сетевыми пакетами. Позволяет описать содержимое пакета на простом языке и сгенерировать плюсовый код с соответстующими полями и методами. Пакеты, описанные в файлах .proto далее называть прокотолом.
3. SFML - библиотека, оборачивающая низкоуровневые, платформозависимые функции в удобные верхнеуровненые. Используем для простой работы с сетью: собранные с помощью пунтка 2 пакеты и отправляем по сети с помощью функций из SFML.
